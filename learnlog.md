1. 在编译有关线程的程序时，要注意链接pthread库

2. linux的新版chrome已经不支持3.17以下的内核，对于我这种使用debian的用户，
果断下载个最新的长期稳定的3.18.10 kernel，编译换上，我的vimium岂能消亡。

```
make menuconfig
make
sudo make modules_install install
```

